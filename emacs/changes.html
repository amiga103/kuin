2012/10/01

kuin-mode.el

2012/08/11

  • == をWarning扱いにしました。 例： if ( a == 0 )
    ※Kuinの等価演算子は == ではなく = です。
  • != をWarning扱いにしました。 例： if ( a != 0 )
    ※Kuinの不等価演算子は != ではなく <> です。
kuin-mode.el

  • コメントのforなどが無視されるようになりました。
  • yasnippetが動作するように修正しました。
