【緩募】KuinでのコンパイルエラーでE0007, E0008, E0025, E0035 の出し方。 参考： Kuinのコンパイルエラー一覧(非公式) dl.dropbox.com/u/9975638/kuin… #Kuin9月 23, 2012

Kuin 0.02～0.023で表示されたエラーをまとめています。
非公式です。
Kuinは、くいなちゃん(@kuina_tesso)が考案したプログラミング言語です。

コンパイルエラー一覧

番号 メッセージ エラーの例 修正方法
E0000 予期せぬエラーが発生しました。コンパイラのバグである可能性が高いです。開発者に問い合わせてください。 
{Kuin 0.023 func型でToStr()}
func Main()
    var a : func<()>
    do Kuin@Dbg(a.ToStr())
end func
( 開発者に問い合わせる )
E0001 コンパイルするソースコードが指定されていません。指定してください。 ソースファイル欄が空欄 ソースファイルを指定する。
E0002 不明なファイル形式です。.knファイルを正しく指定してください。 指定したソースファイルが存在しない。 存在するソースファイルを指定する。
E0003 不明な出力ファイルです。.exeファイルを正しく指定してください。 ・出力ファイル欄に.exeでない拡張子のファイル名を指定。
E0004 不明なアイコンファイルです。.icoファイルを正しく指定してください。 
・アイコンファイル欄に指定したファイルが存在しない。
・.icoでない拡張子のファイル名を指定。
E0005 ファイルが開けませんでした。ファイルが存在し、開ける状態になっているか確認してください。 ソースファイルやimport 文で指定しているファイルが開けない。
E0006 コメントを閉じ忘れています。{ } のようにちゃんと閉じましょう。 
{{ } ←コメントはネストされます。
{ "}" ←文字列中の}ではコメントは終了しません。

{{ }}
{ "}"}
E0007 (不明)
E0008 (不明)
E0009 構文エラーです。ドキュメントを参照し、正しい文法になっているか確認してください。 
・do忘れ。
・ブロックの開始と終わりが不一致の場合もこのエラー
(for文なのにend whileしている場合など)。
・グローバル変数で初期化してしまっている。
・行末に不要な「;(セミコロン)」
・その他
Kuinの構文をKuinの拡張バッカス・ナウア記法 (非公式)などで確認して修正してください。
E0010 その識別子は予約語です。名前を付けるときは、予約語以外を指定してください。 var block : int var block_ : int
E0011 その識別子は既に定義されています。名前を付けるときは、重複しないようにしてください。 
const a : int :: 1
const a : int :: 1
 
const a : int :: 1
const b : int :: 1
E0012 そのファイル名は、標準ライブラリで使われています。ソースファイルには、標準ライブラリで使われていない名前を付けてください。 import Kuin
E0013 クラス定義を閉じ忘れています。end class で閉じましょう。 class C() 
class C()
end class
E0014 列挙体定義を閉じ忘れています。end enum で閉じましょう。 enum E 
enum E
end enum
E0015 ブロックを閉じ忘れています。end文で閉じましょう。 ブロックの開始はあるけれど終わりがない。
E0016 不明な演算子です。ドキュメントを確認してください。 @a
E0017 括弧を閉じ忘れています。( ) のようにちゃんと閉じましょう。 
func Main()
    var a : int :: (1
    )
end func

func Main()
    var a : int :: (1)
end func
E0018 不正な基数です。基数は2〜36が可能で、10や16の場合は表記が特殊です。ドキュメントを確認してください。 
16#FF
10#99

#FF
99
E0019 不正な数値形式です。ドキュメントを確認し、正しい形式で記述してください。 
・2#22
・1.ToStr()

・←2進数は0か1しか使えないため。
・1 .ToStr() { 1と.の間に空白が必要 }
E0020 数値の表現範囲を超えました。大きすぎる値などが指定された可能性があります。 
3^3^3^3
1/0
E0021 Init() 関数の定義が間違っています。func Init(cfg : Kuin@CCfg) としてください。 func Init() func Init(cfg : Kuin@CCfg)
E0022 Main() 関数の定義が間違っています。func Main() としてください。 func Main() : int func Main()
E0023 Release() 関数の定義が間違っています。func Release() としてください。 func Release() : int func Release()
E0024 未定義の識別子です。タイプミスの気をつけ、正しい名前を指定してください。 
・宣言忘れ。
・グローバル変数の「ソースコード名@」忘れ。

("グローバル領域に定義した変数を参照するときは、ソースファイル名@変数名 などのように記述する必要があります。" [出典: var - 自作言語Kuin リファレンスVer.0.03])
E0025 標準ライブラリが見つかりません。ドキュメントを確認し、正しくインストールしてください。 出し方が特殊なので、ノーヒントで出せる人は少ないのではないでしょうか。:-)
E0026 不正なオーバーライドです。親クラスと定義が一致しているか確認してください。 
func FuncName()
    class C1()
        func f()
        end func
    end class
    class C2(C1)
        -*func f()
        end func
    end class
    var tesso : C2
    do tesso.f()
end func

func FuncName()
    class C1()
        func f()
        end func
    end class
    class C2(C1)
        *func f()
        end func
    end class
    var tesso : C2
    do tesso.f()
end func
E0027 オーバーライドする親クラスのメンバが見つかりません。タイプミスに気をつけ、正しい名前を指定してください。 
func Main()
    class C()
        *func f()
        end func
    end class
    var tesso : C
    do tesso.f()
end func

func Main()
    class C()
        func f()
        end func
    end class
    var tesso : C
    do tesso.f()
end func
E0028 親クラスと識別子が重複しています。オーバーライドしたい場合は頭に * を付けてください。 
func FuncName()
    class C()
        func Dtor()
            do Kuin@Dbg("Destructor is called")
        end func
    end class
    var tesso : C
    do tesso.f()
end func
{デストラクタはルートクラスであるKuin@CClassで定義済みなので、オーバーライドになります。} 
func FuncName()
    class C()
        *func Dtor()
            do Kuin@Dbg("Destructor is called")
        end func
    end class
    var tesso : C
    do tesso.f()
end func
E0029 列挙体の値は int型で、コンパイル時に定数にできる必要があります。 
func Main()
    enum E
        a :: 1.2
    end enum
    do Kuin@Dbg((E#a $ int).ToStr())
end func

func Main()
    enum E
        a :: 1
    end enum
    do Kuin@Dbg((E#a $ int).ToStr())
end func
E0030 列挙体の値が重複しています。それぞれの値が重複しないようにしてください。 
func Main()
    enum E
        a
        b
        c :: 1
    end enum
    do Kuin@Dbg((E#a $ int).ToStr())
end func

{:: ～を書かなければ、0から順に割り振られます}
func Main()
    enum E
        a :: -1
        b
        c
    end enum
    do Kuin@Dbg((E#a $ int).ToStr())
end func
E0031 変数の初期値はコンパイル時に定数にできる必要があります。 
func f()
    var b : int :: 1
    const a : int :: b
    do Kuin@Dbg(a.ToStr())
end func

func f()
    const a : int :: 1
    do Kuin@Dbg(a.ToStr())
end func
E0032 型が一致しません。キャストなどを用いて、正しい型に変換してください。 
1+"2"

{ Kuinでは型が厳密に扱われます。 }
1+"2" $ int
E0033 そのメンバはオーバーライドできません。オーバーライドできるのは関数のみです。 *var tesso : int var tesso : int
E0034 そのアクセス指定子は設定できません。アクセス指定子が設定できるのは変数と関数のみです。 -const a : float :: 1 const a : float :: 1
E0035 (不明)
E0036 条件式は bool型でなくてはなりません。 while(1) while(true)
E0037 for文の初期値は int型でなくてはなりません。 
for(0.0, 1)

for(0, 1)
E0038 for文の終値は int型でなくてはなりません。 
for(0, 1.0)

for(0, 1)
E0039 for文の増減値は int型でなくてはなりません。 
for(1, 100, 1.0)

for(1, 100, 1)
E0040 for文の増減値はコンパイル時に定数にできる必要があります。 
for(1, 100, a)

for(1, 100, 1)
E0041 for文の増減値は 0であってはなりません。0以外の値を設定してください。 
for(1, 100, 0)

for(1, 100, 1)
E0042 foreach文に指定できない型です。foreach文に指定できるのは配列やリストなどだけです。 foreach item(1) foreach item("abc")
E0043 try文の条件はコンパイル時に定数にできる必要があります。 catch (tesso) catch (1)
E0044 不正な break文です。break文が作用できるブロックが存在するか確認してください。 
func Main()
    var a : int
    break a
end func
E0045 不正な continue文です。continue文が作用できるブロックが存在するか確認してください。 
func Main()
    if a(true)
        continue a
    end if
end func
{ continue可能なブロックは、while、for、foreachです。 }
E0046 型を指定すべきところに型以外が指定されました。 func f(a : a) func f(a : int)
E0047 不正な参照です。参照先が存在しているか確認してください。 
var tesso : []int
do Kuin@Dbg(tesso.ToStr())

var tesso : []int
do Kuin@Dbg(tesso[0].ToStr())
E0048 不正な左辺値です。代入式の左辺値や@swap演算子の両辺は、代入可能な変数でなくてはなりません。 
do 1 :: 1
{ constで定義された識別子に代入した場合も同様。 }
do tesso :: 1
E0049 @to演算子はここでは使えません。ドキュメントを確認してください。 var a : int :: 5 @to 8
E0050 関数でないものを関数呼び出ししようとしました。 var a : int :: 1()
E0051 配列でないものを配列のように扱おうとしました。 
var tesso : int
do tesso[0] ::1

var tesso : []int
do tesso[0] ::1
E0052 不正なエスケープシーケンスです。\と1文字との組み合わせは特殊な文字を表します。ドキュメントを確認してください。 var a : char :: '\0' { \0(NUL文字)は使用不可。 }
E0053 char型の値にはただ1文字しか指定できません。空もしくは2文字以上が指定されていないか確認してください。 var tesso : char :: 'Kuin' var tesso : []char :: "Kuin"
E0054 do文は副作用のある演算子で終わる必要があります。最後に行う演算が ::演算子や関数呼び出しなどになっているか確認してください。 do tesso = 4 { 代入演算子の間違い } { 代入演算子は=ではなく::です。 }
E0055 長すぎる識別子です。64文字を超える識別子を付けることはできません。 var aaaaaaaaaa…aaaaa : int :: 1 { ←65字はアウト。64字はOK }
E0056 ファイルの書き込みに失敗しました。書き込み先のファイルが既にオープンされていないか確認してください。 { 出力するexeファイルが開かれていて更新不可能。 }
E0057 引数の参照渡しが一致しません。引数を参照渡しするときは & を付ける必要があります。 
func Main()
    func f(a : int)
    end func
    var a : int
    do f(&a)
end func

func Main()
    func f(a : &int)
    end func
    var a : int
    do f(&a)
end func
E0058 参照型を参照渡ししようとしました。引数の参照渡しができるのは値型だけです。 
func Main()
    func g(a : &[]int)
    end func
    var b : []int
    do g(&b)
end func
 
func Main()
    func g(a : []int)
    end func
    var b : []int
    do g(b)
end func
E0059 値を参照渡ししようとしました。引数の参照渡しができるのは変数だけです。 
func Main()
    func f(a : &int)
    end func
    do f(&1)
end func
 定数やfunc型の変数を参照渡した場合も同じエラー。
E0060 不正なファイル名です。ファイル名に使える文字は、半角英数字とアンダーバーのみで、先頭が数字であってはいけません。 識別子として利用可能な文字列をソースファイル名とする。
E0061 Error() 関数の定義が間違っています。func Error(excpt : Kuin@CExcpt) としてください。 func Err() func Err(excpt : Kuin@CExcpt)
E0062
E9999 現段階のコンパイラにまだ実装されていない機能を使おうとしました。今後の実装にご期待ください。 
{Kuin 0.023}
・文字列でswitch
func Main()
    switch("2")
    case ("3")
    end switch
end func

・未実装のメソッド
func Main()
    var a : []char :: "abc"
    var b : []char :: a.Copy()
end func

・配列以外の@in
func Main()
    var a : stack<int>
    var b : bool :: 1 @in a
end func

・配列以外のforeach
func Main()
    var a : stack<int>
    foreach i(a)
    end foreach
end func

・class以外の@new
func Main()
    var a : stack<int> :: @new stack<int>
end func
