2012/08/29

  • [ftdetect] 以前、拡張子の定義（下記）を、_gvimrcに書いて頂くようにしていましたが、ftdetectフォルダにkuin.vimを置くのが正道のようです。
    autocmd BufRead,BufNewFile *.kn set filetype=kuin
    【重要】以前 _gvimrcに上記1行を追加した方は、その行を削除してください。
    代わりに、下記の"kuin.vim (ftdetect)"をftdetectフォルダに置いてください。
    よろしくお願いします。
  • [syntax] gfコマンドを行ったときに、.knファイルも検索するようにしました。set suffixesadd=.knしました。
    こうしないと、import testのtest上でgfしてもファイル移動できません。

2012/08/12

  • [syntax] && || ++ -- をError扱いにしました。
    ※Kuinでは、&& ではなく &、 || ではなく |、 ++ ではなく :+ 1、 -- ではなく :- 1 をそれぞれ用います。
  • [syntax] 10進数の指数表記 (例：314159e-5) をError扱いしないようにしました。 (※実際にKuinでどう扱われるのかは知りません。)
  • [syntax] ToStrなど、組み込みの関数ををシンタックスハイライトするようにしました。
  • [syntax] Kuin 0.02で追加される新しい構文 Test@C#E#Hoge がError扱いされないように修正しました。
  • [syntax] リファクタリングを行いました。

2012/08/11

  • [syntax] == をError扱いにしました。 例： if ( a == 0 )
    ※Kuinの等価演算子は == ではなく、= です。
  • [syntax] != をError扱いにしました。 例： if ( a != 0 )
    ※Kuinの不等価演算子は != ではなく、<> です。
